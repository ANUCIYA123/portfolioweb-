Hospital Management System

Objective: The goal of this project is to digitize and automate hospital operations for better efficiency. It focuses on managing patient records, doctor scheduling, appointments, and billing in one place. Designed using core Java with a modular and object-oriented approach. This project eliminates manual paperwork and improves data accuracy and accessibility. The system is built as a console-based application to demonstrate backend logic and structure. It serves as a foundational project for understanding real-time healthcare data flow.

Tech Used: Java, OOP, File Handling

  • Login system for admin and staff with basic authentication
  • CRUD operations for patient, doctor, and appointment management
  • Automated bill generation and record updates
  • Modular code structure following object-oriented principles
  • Console-based user interface with file-based data handling

Event Ticket Booking System

Objective: This project aims to create an online platform for users to browse, select, and book event tickets. It allows users to register, login, view available events, and make bookings easily. Admins can manage events, track user bookings, and control listings via a dashboard. The system uses Spring Boot for backend and ReactJS for an interactive frontend. Its purpose is to provide a real-world experience in building full stack applications with database integration. It enhances user experience with real-time updates, authentication, and dynamic UI elements.

Tech Used: Java, Spring Boot, ReactJS, PostgreSQL, REST API

  • User authentication and secure registration/login using JWT/session
  • Search, filter, and book event tickets in a responsive UI
  • Admin dashboard for managing events and viewing analytics
  • RESTful API integration with Spring Boot backend
  • Real-time event updates and booking confirmation

Scientific Calculator

Objective: The objective is to develop a fully functional calculator for scientific and general-purpose use. It helps users perform operations like addition, subtraction, trigonometric functions, powers, and roots. Designed for both desktop and mobile devices with a responsive layout. Built using HTML, CSS, and JavaScript to sharpen frontend development skills. This project enhances logical thinking and JavaScript event-handling expertise. It demonstrates the importance of UI/UX in even the simplest web tools.

Tech Used: HTML, CSS, JavaScript

  • Basic arithmetic operations: add, subtract, multiply, divide
  • Scientific functions: sine, cosine, tangent, square root, power
  • Responsive layout for desktop and mobile use
  • Clean and intuitive user interface with instant output
  • JavaScript event handling and real-time computation

EMI Loan Calculator

Objective: The aim of this project is to assist users in calculating EMI and planning financial commitments smartly. It calculates monthly installments based on loan amount, interest rate, and duration. Provides a clean user interface with real-time result updates and financial breakdown. Built using JavaScript and styled with CSS for interactive and responsive design. The goal was to implement math-based logic in a web tool while focusing on usability. It also teaches DOM manipulation and client-side validation in practical scenarios.

Tech Used: HTML, CSS, JavaScript

  • EMI calculation based on principal, rate, and tenure
  • Breakdown of monthly installment, interest, and total amount
  • Dynamic UI with sliders and form inputs
  • Real-time output rendering using JavaScript
  • Helps users in financial planning with clear visuals
